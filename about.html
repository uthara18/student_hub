Project Description

Student Hub is a simple web application developed as part of a web technologies project. It helps students to register, login and access study-related features.

The system allows users to create an account and securely login. After login, users can view notes and use different features available in the system.

This project is built using HTML, CSS, JavaScript, PHP and MySQL.

Technologies Used:

  • HTML - for structure
  • CSS - for styling
  • JavaScript - for validation
  • PHP - for backend
  • MySQL - for database
